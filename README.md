# VaxxVote: Vaccines and Political Party Case Study
This repository contains the materials (data, scripts, reference materials, rubric, and deliverable) to complete the case study.

## Data, Scripts, and Hook Document
The data folder contains the raw data necessary for the case study as csv files. The scripts folder includes helpful code to clean and merge the raw data into a final csv. The hook document folder includes a pdf with the motivation, prompt, and deliverable for the case study. 

## Rubric and Supplemental Materials
The rubric folder contains a pdf that outlines the expectations and goals to successfully complete the case study. The supplemental materials folder includes 2 pdfs with background information on recent immunization trends and the polarization in politics in the U.S. The final pdf provides a helpful guide to understanding linear regression.
